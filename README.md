# Live Tracking App

A real-time location tracking web application built with React, Express, Socket.IO and Leaflet.

## Features

- User to Vendor live tracking
- Real-time location updates using Socket.IO
- Interactive map using OpenStreetMap
- Tracking ID based rooms
- Smooth map animation
- Disconnect detection

## Tech Stack

Frontend:
- React
- Vite
- React Leaflet

Backend:
- Node.js
- Express
- Socket.IO
